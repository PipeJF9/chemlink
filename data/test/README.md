# ChemLink — Test Data

Generic, publicly available structures for validating the docking pipeline.
All files are freely redistributable under their respective source licenses.

---

## Receptors (`receptors/`)

| File | PDB ID | Protein | Resolution | Source |
|---|---|---|---|---|
| `1hsg.pdb` | [1HSG](https://www.rcsb.org/structure/1HSG) | HIV-1 Protease + Indinavir (MK1) | 1.94 Å | RCSB PDB |
| `6lu7.pdb` | [6LU7](https://www.rcsb.org/structure/6LU7) | SARS-CoV-2 Main Protease (Mpro) + N3 inhibitor | 2.16 Å | RCSB PDB |
| `3ptb.pdb` | [3PTB](https://www.rcsb.org/structure/3PTB) | Bovine Trypsin + Benzamidine | 1.70 Å | RCSB PDB |

**Why these three:**
- **1HSG** — the canonical AutoDock benchmark; used in virtually every AutoDock tutorial and paper since 1998. The co-crystallized ligand (MK1/Indinavir) is provided separately.
- **6LU7** — SARS-CoV-2 Mpro, widely used since 2020. The co-crystallized inhibitor (N3/Nirmatrelvir) is included.
- **3PTB** — Bovine Trypsin, smallest and fastest receptor in the set (~1500 atoms). Ideal for rapid CI tests.

> **Note:** PDB files include water molecules and co-crystallized ligands. The docking pipeline's preparation step (MGLTools) strips these automatically. For manual preparation, use `chemlink doctor` to verify your environment before processing.

## Quick test commands

```bash
# Minimal test — single receptor, single ligand (fastest)
chemlink docking \
  --receptor data/test/receptors/3ptb.pdb \
  --ligands  data/test/ligands/benzamidine.sdf \
  --output   data/output/test_3ptb

# Canonical benchmark — HIV-1 Protease
chemlink docking \
  --receptor data/test/receptors/1hsg.pdb \
  --ligands  data/test/ligands/indinavir.sdf \
  --output   data/output/test_1hsg

# Library test — 6 ligands against HIV-1 Protease
chemlink docking \
  --receptor data/test/receptors/1hsg.pdb \
  --ligands  data/test/ligands/library_6.sdf \
  --output   data/output/test_library
```
